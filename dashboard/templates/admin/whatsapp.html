{# ── Command Reference ────────────────────────────────────────────────── #}

💬 WhatsApp Command Reference

Send any of these commands from +{{ admin_number }} to VCPilot on WhatsApp once connected:

STATUS

System status overview, active mode, and count of open positions/signals.

POSITIONS

List all open positions with current size, entry price, stop price, and live P&L.

SIGNALS

List all stock signals generated by the screener for today with entry levels.

WATCHLIST

Show the tickers currently in the system's watchlist.

MARKET

Get the current evaluated market regime status (BULL, CAUTION, or BEAR).

PAUSE / RESUME

Suspend or resume placing new trade entries from signals.

REPORT

Generate and send the daily trading report summary.

SKIP <TICKER>

Cancel today's signal for the ticker (e.g. SKIP BHP).

EXIT <TICKER>

Trigger emergency exit of a position on next market open (e.g. EXIT FMG).

STOP <TICKER> <PRICE>

Update the stop loss price for an open position (e.g. STOP BHP 41.50).

RULE <RULE_ID> ON|OFF

Globally enable or disable a trading rule (e.g. RULE trend_template_1 OFF).

HELP

Fetch the quick help list of all available commands.
