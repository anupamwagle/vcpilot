How Exchange Management Works

Step 1 — Super Admin (here): Enable exchanges that VCPilot supports. Equity exchanges (ASX, NYSE, NASDAQ) route via IBKR. Crypto exchanges route via the ccxt library.

Step 2 — Org Admin (/admin/config): Set active_exchanges to comma-separated exchange keys (e.g. ASX,NYSE). Only enabled exchanges can be activated.

Step 3 — Trader (/watchlist): Add stocks/crypto via the exchange selector. VCPilot fetches price history on-demand and screens the instrument against Minervini rules.

Crypto only: Org admins must also configure crypto_api_key and crypto_api_secret in /admin/config for live crypto trading. Fundamentals screening is skipped for crypto assets.
